# Contributing

This project is an experimental reference kit. Contributions should keep the scope narrow and make GNAP/OpenAPI behavior easier to inspect.

## Local Checks

```sh
pnpm install
pnpm test
pnpm build
```

## Guidelines

- Keep examples compact and readable.
- Add tests for validation behavior.
- Use placeholder URLs and local fixtures only.
- Do not add real credentials or live payment API calls.
- Describe new extension behavior as experimental unless it has been formally adopted elsewhere.
